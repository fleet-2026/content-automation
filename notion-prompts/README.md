## Format

Each guide is organized into themed sections. Every prompt is an `###` entry and follows a
Fill-in-the-blank / Question-based / Examples format:

- **Fill-in-the-blank prompts** use `[bracketed]` placeholders you replace with your own details for structured, ready-to-use output.
- **Question-based prompts** help you explore a topic or go deeper.
- **Examples** show how a filled-in prompt looks in practice.

Copy any prompt, swap in your details, and paste it into Claude (free or paid).

## Importing into Notion

Option A — Import menu:

1. In Notion, open the page where you want the content (or create a new page).
2. Click the `•••` menu in the top-right.
3. Choose **Import**.
4. Select **Text & Markdown**.
5. Pick the `.md` file(s) you want — e.g. `00-All-Prompts-for-Claude-MASTER.md` for everything, or individual guides.

Option B — Drag and drop:

- Drag a `.md` file directly onto a Notion page to import it.

Notion converts each Markdown heading into a heading block, so the section structure and the `###` prompt entries become a navigable outline.
